# Pre-Launch Checklist — Make It Media

Work through top to bottom. Check off as you go.

---

## 🟢 Phase 1 — Must-Have Before First Social Post

### Email Platform + Delivery
- [ ] Choose email platform (ConvertKit/Kit, MailerLite, or Beehiiv)
- [ ] Connect domain / verify sender email
- [ ] Set up brand template (logo, colours, footer)
- [ ] Build lead magnet delivery email (instant auto-send on signup)
- [ ] Write welcome sequence — 3–5 emails over 7–14 days
  - [ ] Email 1: Deliver lead magnet + set expectations
  - [ ] Email 2: Story / credibility
  - [ ] Email 3: Value / quick win
  - [ ] Email 4: Case study or social proof
  - [ ] Email 5: Soft CTA to book strategy call

### Lead Magnets + Landing Pages
- [ ] Decide on 1–3 higher-ticket lead magnets (topics + formats)
- [ ] Create lead magnet 1 (PDF/guide/template)
- [ ] Create lead magnet 2
- [ ] Create lead magnet 3
- [ ] Build landing page 1 on site (form connected to email platform)
- [ ] Build landing page 2
- [ ] Build landing page 3
- [ ] Test: submit form → email arrives → magnet delivers

### ManyChat + Social DM Automation
- [ ] Set up ManyChat account + connect Instagram
- [ ] Build comment-to-DM flow for lead magnet 1
- [ ] Build comment-to-DM flow for lead magnet 2
- [ ] Build comment-to-DM flow for lead magnet 3
- [ ] Flow captures email before sending magnet
- [ ] LinkedIn DM template (manual or via Aware — ManyChat doesn't integrate directly)
- [ ] Test each keyword trigger end-to-end

### Booking System
- [ ] Set up Calendly / SavvyCal / Cal.com
- [ ] Connect Google Calendar
- [ ] Configure strategy call event (duration, buffer, availability)
- [ ] Add intake questions to booking form
- [ ] Set up confirmation + reminder emails
- [ ] Embed on website (contact page + CTAs)
- [ ] Add link to email signature
- [ ] Add link to welcome sequence final email

### CRM / Lead Tracking
- [ ] Set up Airtable or Notion lead database
- [ ] Fields: name, email, source, lead magnet, date, status, notes
- [ ] Connect email platform → CRM (Zapier/native)
- [ ] Connect booking system → CRM

### Analytics + Tracking
- [ ] GA4 installed + firing on all pages
- [ ] Event tracking on all CTAs (use existing tracking IDs per CLAUDE.md)
- [ ] Meta Pixel installed
- [ ] LinkedIn Insight Tag installed
- [ ] UTM link template created for social posts
- [ ] Conversion events configured in GA4 (form submit, booking)

---

## 🟡 Phase 2 — Month 1–2 (After Posting Starts)

- [ ] Payment system — only if productising (Stripe or Dubsado/HoneyBook)
- [ ] Client onboarding automation (forms, contracts, welcome pack)
- [ ] Retargeting audiences built (Meta + LinkedIn)
- [ ] Testimonial collection system (Senja, or simple form + process)
- [ ] Second lead magnet sequence live
- [ ] A/B test headlines on top-performing landing page

---

## 🔴 Later — Don't Build Yet

- HubSpot paid / ActiveCampaign full suite
- Website chatbot
- Affiliate / referral program
